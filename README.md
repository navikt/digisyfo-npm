digisyfo-npm
================

Felles kode for digitalisering av sykefraværsoppfølgingen.

# Komme i gang

`npm install`

Kjør tester:

`npm test`

Kjøre tester kontinuerlig under utvikling. 

`npm run watch`

Se ellers flere kommandoer i `package.json`.
